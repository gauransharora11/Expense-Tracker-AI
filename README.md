AI / ML Project – Expense Categorization System

💼 Expense Tracker AI

An AI-powered Expense Tracker that automatically categorizes expenses (Food, Travel, Shopping, Entertainment, Other) using Machine Learning and a Streamlit Web UI.

The model is trained from CSV datasets and can be retrained automatically whenever data changes.

🚀 Features

✅ Expense category prediction using ML

✅ Auto-trains model from CSV datasets

✅ Classifies unknown inputs as other

✅ Clean and interactive web UI (Streamlit)

✅ Manual retrain button

✅ Modular project structure (industry-style)

🧠 Categories Supported

🍔 Food

🚕 Travel

🛍 Shopping

🎬 Entertainment

❓ Other (for unclear inputs)

🛠 Tech Stack

Python

Scikit-learn

Pandas

Joblib

Streamlit

TF-IDF + Logistic Regression
